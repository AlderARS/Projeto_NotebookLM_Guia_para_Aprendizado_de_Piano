# 🎹 NotebookLM – Guia para Aprendizado de Piano

## 📌 Sobre o Projeto

Este projeto foi desenvolvido utilizando o Google NotebookLM com o objetivo de criar um assistente de estudos especializado no aprendizado de piano para iniciantes e estudantes de nível intermediário.

A proposta foi centralizar conteúdos de diferentes fontes confiáveis, permitindo gerar planos de estudo personalizados, recomendações de exercícios, explicações teóricas, sugestões de repertório e apoio na evolução técnica do aluno.

Além de servir como um repositório de conhecimento, o Notebook foi projetado para atuar como um tutor virtual capaz de responder dúvidas específicas e criar roteiros de aprendizagem adaptados ao nível do estudante.

---

# 🎯 Objetivos do Estudo

Os principais objetivos deste projeto foram:

* Criar uma base de conhecimento especializada sobre piano e teclado.
* Avaliar a capacidade do NotebookLM em consolidar conteúdos de múltiplas fontes.
* Gerar planos de estudo personalizados para iniciantes.
* Testar a geração de resumos, recomendações e materiais de apoio.
* Avaliar a qualidade das respostas produzidas pela IA em cenários reais de aprendizado musical.
* Identificar limitações e oportunidades de melhoria na construção de bases temáticas.

# 🧠 Engenharia de Prompts e Cicatrizes

## Desafio Encontrado

Durante os testes foi identificado que o limite de fontes disponíveis para upload no NotebookLM restringe parcialmente a profundidade do conhecimento gerado.

Embora playlists completas tenham sido utilizadas para ampliar a quantidade de conteúdo disponível, alguns resultados apresentaram limitações, principalmente na geração automática de apresentações, vídeos e recursos visuais.

Apesar disso, o Notebook demonstrou excelente desempenho na criação de planos de estudo, explicações teóricas e recomendações práticas.

---

# 📚 Glossário

| Conceito   | Definição                                                                |
| ---------- | ------------------------------------------------------------------------ |
| Dedilhado  | Sistema de numeração dos dedos utilizado para executar trechos musicais. |
| Legato     | Execução suave e conectada das notas.                                    |
| Staccato   | Execução curta e destacada das notas.                                    |
| Pentagrama | Conjunto de cinco linhas utilizado para escrever música.                 |
| Clave      | Símbolo que determina a altura das notas.                                |
| Dó Central | Principal referência de localização no teclado.                          |
| Compasso   | Organização dos tempos dentro de uma música.                             |
| Acorde     | Conjunto de três ou mais notas tocadas simultaneamente.                  |
| Arpejo     | Execução sequencial das notas de um acorde.                              |
| Fermata    | Indicação de prolongamento de uma nota.                                  |

# 🔄 Prompts Reutilizáveis

### Fundamentos

* Como identificar o Dó Central no teclado?
* Como funcionam as claves de Sol e Fá?
* Quais são os erros mais comuns de postura para iniciantes?

### Técnica

* Explique o sistema de numeração dos dedos.
* Como desenvolver independência entre as mãos?
* Quais exercícios ajudam na agilidade dos dedos?

### Teoria Musical

* O que significa uma fórmula de compasso 3/4?
* Qual a diferença entre sustenido, bemol e bequadro?
* Para que servem as linhas suplementares?

### Repertório

* Quais músicas são recomendadas para iniciantes?
* Como estudar uma partitura pela primeira vez?
* Qual o melhor método para aprender uma música nova?

# 🎓 Principais Aprendizados

Este projeto demonstrou que o NotebookLM pode atuar como uma poderosa ferramenta de aprendizado personalizado quando alimentado com fontes de qualidade.

Os testes mostraram bons resultados na criação de planos de estudo, organização de conteúdo, recuperação de informações e explicação de conceitos musicais. Também evidenciaram limitações relacionadas à quantidade de fontes e geração de recursos visuais, fornecendo importantes aprendizados para futuras evoluções da base de conhecimento.
